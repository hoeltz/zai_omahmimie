## 🚨 **Troubleshooting**

### **Export Tidak Berjalan:**

#### **1. Check Console Errors:**
```javascript
// Buka browser console (F12)
// Cari error messages:
// - Network errors
// - JavaScript errors
// - API response errors
```

#### **2. Check API Response:**
```bash
# Test API langsung:
curl http://localhost:3000/api/properties/export/pdf

# Should return PDF binary data, bukan error JSON
```

#### **3. Check Database:**
```bash
# Pastikan ada data:
curl http://localhost:3000/api/properties

# Response harus array dengan > 0 items
```

#### **4. Check Browser Download:**
- Chrome: Check download bar
- Firefox: Check download history
- Safari: Check downloads folder
- Mobile: Check download folder/notification

### **Common Issues & Solutions:**

#### **Issue: "Gagal mengunduh PDF"**
**Cause:** API error atau network issue
**Solution:**
1. Check internet connection
2. Refresh browser
3. Check browser console
4. Test API endpoint langsung

#### **Issue: File kosong (0 bytes)**
**Cause:** Tidak ada data di database
**Solution:**
1. Tambah data properti dulu
2. Test dengan data yang ada
3. Check database connection

#### **Issue: Download tidak otomatis**
**Cause:** Browser blocking download
**Solution:**
1. Check browser download settings
2. Allow automatic downloads
3. Try manual download dari API

#### **Issue: Error di production (Vercel)**
**Cause:** Environment variables atau configuration
**Solution:**
1. Check Vercel function logs
2. Verify build success
3. Check database connection
